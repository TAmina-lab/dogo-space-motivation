# Dogo’s Space Motivation 🐶🚀

A simple Scratch project that provides motivational messages for beginner programmers.

## How to use
- Open the project in Scratch
- Click on Dogo to receive motivation
- Pet Dogo to increase your friendship level 💖

## About
This project was inspired by my own experience of struggling to start learning programming.
